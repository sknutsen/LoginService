# Env variables:  
DB_HOST - IP of Postgres server  
DB_USER - Username  
DB_PASS - Password  
ACCESS_TOKEN_SECRET - Secret for signing JWT access token  
REFRESH_TOKEN_SECRET - Secret for signing JWT refresh token
